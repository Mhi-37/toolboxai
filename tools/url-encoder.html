Encodeur URL percent-encoding

L'encodage URL (percent-encoding) convertit les caractères spéciaux en séquences %XX pour qu'ils soient valides dans une URL. Les espaces deviennent %20, les accents et symboles sont encodés. Indispensable pour construire des query strings, des liens avec paramètres, ou transmettre des données dans une URL. Le décodeur fait l'inverse. Utilisez encodeURIComponent pour encoder une valeur de paramètre, ou notre outil pour des conversions manuelles. Les développeurs web l'utilisent quotidiennement pour les APIs et formulaires.
