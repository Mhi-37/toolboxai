Numérotation de lignes

Ajoutez des numéros de ligne à un texte pour faciliter les références croisées, le partage de code ou les corrections. Format : "1 | première ligne". Pratique pour les révisions de documents, le debugging ou les tutoriels. Utile lorsque vous devez indiquer "voir ligne 42" à un collègue.
