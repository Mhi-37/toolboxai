Décodeur de token JWT


Un JWT (JSON Web Token) est composé de 3 parties base64url : header, payload, signature. Notre outil décode le header et le payload en JSON lisible. Attention : aucune vérification de signature n'est effectuée. Utilisez pour inspecter le contenu d'un token (claims, expiration). Ne jamais partager des tokens contenant des données sensibles. Les développeurs l'utilisent pour déboguer l'authentification.
