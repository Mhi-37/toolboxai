Générateur de dégradés pour le web

Créez des dégradés CSS (linear-gradient) visuellement. Choisissez deux couleurs et un angle, le code CSS est généré. Utilisez-le pour les arrière-plans, boutons, headers. La propriété background ou background-image accepte linear-gradient(angle, color1, color2). Copiez et collez dans votre feuille de style.
