Générateur de balises meta pour le SEO

Créez les balises meta essentielles pour le référencement : title, meta description, Open Graph (Facebook, LinkedIn) et Twitter Cards. Ces balises contrôlent l'affichage dans les résultats Google et sur les réseaux sociaux. Un bon title fait 50-60 caractères, une description 150-160. Copiez le code généré dans le <head> de vos pages. Indispensable pour tout site web professionnel.
