Encodeur d'entités HTML

Les entités HTML permettent d'afficher des caractères spéciaux sans qu'ils soient interprétés : < devient &lt;, > devient &gt;, les guillemets et accents aussi. Essentiel pour afficher du code HTML dans une page, éviter les injections XSS, ou gérer des caractères Unicode. Notre outil encode tous les caractères non-ASCII en entités numériques ({) et décode les entités nommées (&amp;) et numériques. Indispensable pour les développeurs web.
