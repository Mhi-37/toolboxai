Détecteur User-Agent

Le User-Agent est une chaîne envoyée par le navigateur à chaque requête HTTP. Il indique le navigateur, la version, le système d'exploitation. Utilisé pour l'analytics, l'adaptation de contenu ou le débogage. Notre outil affiche le vôtre. Les sites peuvent détecter mobile/desktop à partir de cette info.
