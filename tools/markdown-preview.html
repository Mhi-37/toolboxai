Prévisualisation Markdown

Le Markdown est une syntaxe légère pour formater du texte. # pour les titres, ** pour le gras, * pour l'italique, - pour les listes, [texte](url) pour les liens. Notre outil convertit en HTML basique et affiche l'aperçu. Utile pour rédiger des README, des commentaires ou des articles. Conversion simplifiée côté client.
