Bloc-notes sauvegardé localement

Prenez des notes qui restent sur votre appareil. Le localStorage du navigateur conserve vos données entre les sessions. Aucune donnée n'est envoyée à un serveur. Idéal pour des notes rapides, listes, idées. Les données sont liées à ce domaine et à ce navigateur. Effacez le cache du site pour supprimer les notes.
