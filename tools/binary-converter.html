Convertisseur de bases numériques

Le binaire (base 2) utilise uniquement 0 et 1, c'est le langage des ordinateurs. L'hexadécimal (base 16) utilise 0-9 et A-F, très pratique pour les couleurs CSS et les adresses mémoire. Le décimal (base 10) est notre système usuel. Convertissez facilement entre ces bases pour le développement, l'électronique ou l'apprentissage. Entrez une valeur dans n'importe quel champ et les autres se mettent à jour. Les développeurs l'utilisent pour le débogage et la manipulation de bits.
