Calculateur de différence de dates

Calculez le nombre de jours entre deux dates. Utile pour les échéances, les durées de projet, les âges ou les périodes. Le calcul utilise les dates complètes (jour, mois, année). Résultat en jours. Si la date 2 est avant la date 1, le résultat est négatif.
