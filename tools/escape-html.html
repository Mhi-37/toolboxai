Échappement HTML pour la sécurité

Échappez les caractères < > & " ' pour afficher du contenu utilisateur en toute sécurité et éviter les injections XSS. Convertissez < en &lt;, etc. Déséchapper fait l'inverse. Essentiel pour les applications web qui affichent des entrées utilisateur.
