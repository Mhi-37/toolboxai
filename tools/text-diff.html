Comparateur de texte (diff)


Comparez deux versions d'un texte pour identifier les modifications. Utile pour vérifier des corrections, comparer des traductions, ou auditer des changements de contrat. Notre outil compare ligne par ligne : les lignes identiques, ajoutées ou supprimées sont identifiées. Version simplifiée d'un diff style Unix. Idéal pour les rédacteurs, traducteurs et toute personne travaillant sur des documents. Pas besoin d'outil en ligne de commande, tout dans le navigateur.
